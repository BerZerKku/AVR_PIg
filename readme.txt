Меню для АВАНтов в СТАРОМ исполнении (4-х строчный символьный ЖКИ и 16-и кнопочная клавиатура).
Прогрмамма универсальная. Изначально с блока БСП считывается текущая версия аппарата и затем происходит настройка меню.
Изначально была сделана на базе прошивки для нового исполнения АВАНТ.
